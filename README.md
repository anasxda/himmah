# همّة | Himmah

تطبيق ويب لمتابعة تمارين الأسبوع (٧ أيام) بنظام تدريب بوش/بول/ليقز (Push/Pull/Legs). تطبيق واحد بملف واحد — بدون خادم أو تثبيت.

## المزايا
- جدول أسبوعي (٧ أيام) مع تصنيف كل يوم (دفع/سحب/أرجل/راحة) قابل للتغيير.
- لكل تمرين: الاسم (مع قائمة اقتراحات جاهزة)، الجلسات، العدّات، الوزن (كجم/رطل)، وملاحظات.
- اقتراح آلي لوزن الأسبوع القادم بناءً على الأداء المُسجَّل (مبدأ التحميل التدريجي).
- إنهاء اليوم يعرض عبارة تحفيزية عشوائية.
- مؤقت راحة بين المجموعات (٣٠ إلى ١٨٠ ثانية) مع تنبيه صوتي.
- شريط تاريخ ووقت حي مرتبط بساعة الجهاز.
- دعم كامل للغتين العربية والإنجليزية (مع تبديل الاتجاه RTL/LTR).
- وضع ليلي/نهاري.
- كل البيانات تُحفظ محليًا في متصفحك (localStorage) — لا يوجد خادم أو قاعدة بيانات.

## التشغيل
افتح `index.html` مباشرة في أي متصفح، أو فعّل GitHub Pages على هذا المستودع (سيُعرض تلقائيًا لأن اسم الملف `index.html`).

## ملاحظة عن الخط المستخدم
الواجهة تستخدم خط **ثمانية** (Thmanyah Sans / Thmanyah Serif Display) مُضمَّنًا داخل ملف `index.html` كـ base64.

**تنبيه ترخيص:** بحسب رخصة الخط الرسمية (thmanyah Font License)، التضمين مسموح فقط كجزء من منتج "مُجمَّع أو معبّأ أو مُبهَم" (compiled/packaged/obfuscated)، وتمنع الرخصة صراحة رفع أو استضافة أو إتاحة الخط بشكل يسمح لأي طرف باستخراجه كملف خط مستقل — وتضمينه كنص base64 خام وقابل للقراءة داخل كود مصدري مرفوع على GitHub لا يستوفي شرط "obfuscated" بدقة، ويقابل حالات محظورة صراحة في الرخصة (رفع/استضافة/تمكين الاستخراج).

تم إبقاء الخط في هذه النسخة بقرار من مالك المشروع مع العلم بهذه المخاطرة. إذا احتجت لاحقًا نسخة متوافقة تمامًا مع الرخصة، الخيار الأسلم هو إزالة تضمين الخط والاعتماد على خطوط النظام (سلسلة `-apple-system, 'Segoe UI', 'Noto Sans Arabic', Tahoma`), أو التواصل مع thmanyah (ask@thmanyah.com) لطلب إذن استخدام موسّع.
